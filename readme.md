Task Tracker Application

📌 Overview

The Task Tracker is a web-based application designed to help users manage and track their daily tasks efficiently. It provides a seamless experience with user authentication, task management, and real-time updates.

✨ Features

User Authentication (JWT-based login/signup)

Create, Update, Delete tasks

Task Categorization (Completed, Pending, In Progress)

Secure REST API

Responsive UI

Backend with Spring Boot and PostgreSQL

🛠️ Tech Stack

Frontend:

React.js

React Router

Tailwind CSS

Backend:

Spring Boot

Spring Security (JWT authentication)

PostgreSQL

JPA/Hibernate
